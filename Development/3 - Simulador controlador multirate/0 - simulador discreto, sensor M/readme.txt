Comparação das seguintes situações:

- Controlador contínuo, sensor contínuo de velocidade
- Controlador discreto, sensor discreto de velocidade (método M)

O controlador contínuo foi discretizado para um tempo de amostragem Ts
literal (no mathematica), portanto seus ganhos são calculados fora do bloco
discrete transfer function.
